The Status desktop app is the best way to experience Status on Windows. Take a look at the steps below to get started.

<Admonition type="info">
  To protect yourself from malware, viruses or scams, always download the Status app from a trusted source. For the Windows version, use the [Status website](https://status.app)<ExternalIcon />.
</Admonition>

## Download the Status app

1. Visit the [Status website](https://status.app)<ExternalIcon />.
1. Click <ContextTag icon={<WindowsIcon />}>Download for Windows</ContextTag>.
1. Once the download is complete, find the Status app installer. This is typically located in your Downloads folder.
1. Double-click the Status .exe file.
1. To launch the Status app, open the Applications folder and double-click the Status icon.

## If you can't download the Status desktop app on Microsoft Edge

Microsoft Edge for Windows includes the SmartScreen feature, which can prevent you from downloading the Status desktop app. If you receive a message that says "StatusIm-Desktop.exe isn't commonly downloaded" follow these steps:

1. Hover over the Microsoft Edge warning message and click <OptionsIcon /> **More options**.
1. Select **Keep**.
1. On the **Make sure you trust StatusIm-Desktop** screen, click **Show more**.
1. Click **Keep anyway**.

<Admonition type="info">
  The Status desktop app you download from the Status website ([https://status.app](https://status.app)) is safe and free from malware.
</Admonition>

## Common questions

### Do I need administrator privileges to install the Status app?

Yes, you need administrator privileges to install Status desktop. This is to make sure the installation is correctly set up.

### How can I ensure that I am downloading the latest version of the app?

Always visit the [Status website](https://status.app)<ExternalIcon /> to download the latest version of the app. When running Status desktop, the app automatically updates to the most recent version.

### What should I do if the app does not open after installation?

Verify that your system meets the minimum app's requirements. Consider restarting your computer and [reinstalling](./uninstall-status-desktop.mdx) if the issue persists or the installation fails.
