The [Keycard](./keycard-your-quick-start-guide.mdx) hardware wallet provides various functionalities when you use it with the Status app. These functionalities differ depending on whether you use the mobile or desktop version of the app.

## Keycard functionality by platform

Here you can find the differences in Keycard functionality for Status app release 2.32.

<Admonition type="tip">
  To find the version of the Status app you are using, see [Check your Status app version](../getting-started/check-your-status-app-version.mdx).
</Admonition>
