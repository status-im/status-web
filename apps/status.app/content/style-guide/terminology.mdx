## Word choice

Write in plain English.

- Don't use words that can be confusing for non-native speakers. For more details, see the [A-Z of alternative words](https://www.plainenglish.co.uk/the-a-z-of-alternative-words.html) composed by the Plain English Campaign.
- Don't use Latin abbreviations like *e.g.*, *etc.* or *i.e.* Use an English equivalent, like *for example*, *and so on* or *that is*.
- Don't use different forms of the same word in a single sentence. For example, instead of using "The community Owner *owns* the tokens," use "The community Owner *holds* the tokens."

## Status product and feature names

Products and names include:

- Status apps features (for example, Status Communities).
- Products and services that are part of the [Status Network <ExternalIcon />](https://statusnetwork.com/)(for example, Keycard).
- Status apps features or proper nouns (like the Activity Center).

<Admonition type="note">
  Observe the capitalization convention. For example, use *Activity Center* and not *activity center*.
</Admonition>
